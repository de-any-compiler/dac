Usage: dac <input-binary> [options]

dac lifts ELF / PE / Mach-O binaries to high-level source. See the
requirements specification (§10.1) for the source of truth on these flags;
most are parsed today and become active milestone by milestone.

Reconstruction level:
  -O0                       Faithful annotated listing of lifted IR.
  -O1                       Best-effort low-level reconstruction.
  -O2                       Structured reconstruction with idioms.
  -O3                       AI-assisted semantic reconstruction.

Input:
      --arch <arch>         Override the auto-detected architecture.
      --format <fmt>        Input format: elf, pe, mach-o, or auto
                            (default: auto).
      --plugin <path>       Load a dynamic analysis or backend plugin.
      --hints <path>        Apply user-supplied function-signature /
                            struct hints from a TOML file (FR-20).

Output:
      --target <lang>       Target language: c or cpp (default: c).
      --output <path>       Write reconstructed source to <path>.
      --emit-ir             Export intermediate representation artifacts.
      --emit-cfg            Export control-flow graphs (DOT).
      --emit-report         Export the analysis report.
      --emit-annotations    Export annotations alongside the source.
      --callgraph           Export the whole-program call graph (DOT).
      --xrefs <subject>     Print xrefs for <subject> (symbol or hex VA).
      --hide-crt            Collapse CRT / startup function bodies to
                            forward declarations (FR-21).

AI:
      --no-ai               Disable all AI providers.
      --ai-provider <name>  Use AI provider <name>.
      --ai-strict           Drop AI deltas that would overwrite an
                            Observed fact (spec §13.5).

Execution:
      --deterministic       Reject pipelines that register a
                            non-deterministic pass (NFR-9).
      --threads <n>         Limit analysis to <n> threads.

Diagnostics:
      --json                Emit machine-readable JSON diagnostics.
      --debug               Verbose traces (defaults filter to debug
                            level when RUST_LOG is unset).

Meta:
  -V, --version             Print dac version and build id and exit.
  -h, --help                Print this message and exit.
